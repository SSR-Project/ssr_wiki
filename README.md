### アプリケーション構成

>#####管理者用アプリケーション (ssr_adming) 
>* 管理者用EmailとPASSで認証
>* 全てのユースケースが可能
>
>#####在学生と修了学生用アプリケーション (ssr_student)
>* 学内EmailとPASSで認証
>
>##### 非学生と在学生用アプリケーション (ssr_public)
>* EmailとPASSで認証 
>* 説明会関連のユースケースを行う。
